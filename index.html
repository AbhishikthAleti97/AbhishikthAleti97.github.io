Manufacute automater

A desktop application to automate manufacturing process

This application aims to automate manufacturing processes by taking a CAD file as input.

This application was based on text parsing, where shapes were discovered from the CAD models. After analyzing the shapes and inputs, the application decides on what processes the product needs to undergo before obtaining the final product. Based on the processes required, the application suggests places nearby the user, where the processes can be carried out. This application was made using Tkinter and Python.
